Frequently Asked Questions (FAQ)

  • How do I upload a file?

    Navigate to the Upload File page. Click the "Choose File" button, select your data file (CSV, XLSX, or JSON), and then click "Upload." Ensure the file follows the correct format for accurate analysis.

  • What file types are supported?

    Supported file types include:

    • CSV (Comma-Separated Values)
    • XLSX (Excel Files)
    • JSON (JavaScript Object Notation)
    Files must contain structured data with headers in the first row.

  • How do I perform predictive modeling?

    Navigate to the Predictive Modeling page. Select the target column, choose the features, specify the model type (e.g., Linear Regression, Decision Tree, or SVM), and click "Run Analysis." The system will process your data and provide results, including predictions and error metrics.

  • What features does the app provide?

    Key features include:

    • Data Upload and Preprocessing
    • Predictive Modeling (Regression and Classification)
    • Data Visualization
    • Customizable Reports
    • Time-Series Analysis

  • How can I enable Two-Factor Authentication (2FA)?

    Go to the Enable 2FA page under your account settings. Follow the instructions to link your authentication method.

  • What if I encounter an error?

    If you encounter any issues:

    • Ensure your file is in a supported format and has a valid structure.
    • Refer to tooltips provided on pages for guidance.
    • If issues persist, contact support at support@dataanalysisapp.com.

  • How secure is my data?

    Your data is securely processed and stored. We follow best practices to ensure data privacy and integrity. Uploaded files are not shared with third parties.

  • How can I access detailed reports?

    Navigate to the Reports page. You can view summaries and download detailed reports in various formats.
